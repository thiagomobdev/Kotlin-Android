## Resumo 1 (#8)

Inverta os dígitos de um número decimal usando um loop `while`. Um `var worker` auxiliar irá ajudar você a produzir iterativamente o `result`:

| Worker     | Result |
|------------|--------|
| 1234       | 0      |
| 123        | 4      |
| 12         | 43     |
| 1          | 432    |
| 0          | 4321   |
