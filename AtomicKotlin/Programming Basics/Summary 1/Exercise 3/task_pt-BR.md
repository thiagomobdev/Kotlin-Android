## Resumo 1 (#3)

Escreva uma função `first()` que tenha um *corpo de expressão* começando com `if`. A função recebe três parâmetros `Boolean` e os combina com "and" para produzir o resultado. Escreva `second()` para fazer o mesmo, mas combinando todos os três parâmetros com "or". No `main()`, exiba o resultado da combinação de `first()` e `second()` usando tanto "and" quanto "or":

| Argumentos `first()` | Argumentos `second()` |
|----------------------|-----------------------|
| true, true, true     | false, false, false   |
| true, false, true    | false, true, false    |
